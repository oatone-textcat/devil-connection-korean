# 데빌 커넥션 비공식 한글 번역 패치

본 한글 패치는 Nyabi님의 데빌 커넥션 한글 패치에 기반하여 번역 팀 글냥이 소속원들(Oatone, 체퓨)의 도움을 받아 제작되었습니다.

> **면책 조항**
>
> - 본 프로젝트는 **비공식 팬 번역**으로, 원작자 및 배급사와 어떠한 제휴·승인 관계도 없습니다.
> - 본 프로젝트는 게임 본편 또는 이를 대체할 수 있는 데이터를 제공하지 않습니다.
> - 배포되는 파일은 번역 적용을 위한 수정 데이터에 한정되며 단독으로 실행될 수 없습니다.
> - 패치를 사용하려면 사용자가 별도로 정품 게임을 보유하고 있어야 합니다.
> - 원본 게임의 저작권 및 모든 권리는 해당 권리자에게 있습니다.
> - 본 번역물은 원작의 파생 저작물에 해당하며, 원작의 이용 약관 및 관련 법률의 적용을 받을 수 있습니다.
> - 본 번역의 사용으로 발생하는 문제에 대한 책임은 사용자에게 있습니다.

---

## 설치 방법

1. GitHub 페이지 상단의 **[Releases](../../releases)** 탭으로 이동합니다.
2. 최신 릴리즈에서 본인의 운영체제에 맞는 패치 프로그램을 다운로드합니다.
   - Windows: `DevilConnection-Patcher-Windows-x64.exe`
   - macOS: `DevilConnection-Patcher-macOS-arm64.zip`
   - Linux: `DevilConnection-Patcher-Linux-x86_64`
3. 다운로드한 프로그램을 실행하고 안내에 따라 진행합니다.
   - Linux: 실행 파일에 실행 권한을 부여해야 합니다: `chmod +x DevilConnection-Patcher-Linux-x86_64`
4. 게임을 실행하면 한글 번역이 적용됩니다!

## 주의사항

- 게임 업데이트 시 수정 사항이 초기화될 수 있습니다.
- 원본 파일은 반드시 백업해두세요.
- 본 번역 패치 사용으로 인한 게임 오류나 문제는 책임지지 않습니다.
- macOS에서 게임 실행 시 보안 경고가 나타날 수 있습니다. 시스템 설정 > 개인정보 보호 및 보안에서 허용해주세요.

## 번역 기여 안내

- 오역, 의역, 번역 누락 등 문제가 되는 번역을 발견하시면 GitHub 이슈로 제보해주시거나, 아래 커뮤니티 또는 디스코드를 통해 연락해주시면 감사하겠습니다.

### 서체(폰트) 사용 안내

본 프로젝트는 가독성 향상을 위해 각 권리자가 별도로 배포하는 서체 파일들을 포함하고 있습니다.

각 서체의 저작권은 해당 권리자에게 있으며 본 프로젝트는 서체의 소유권을 주장하지 않습니다.
서체의 사용 조건은 각 권리자가 제공하는 이용 약관 및 라이선스를 따릅니다.

각 서체의 저작권 고지 및 라이선스 전문은
`THIRD_PARTY_LICENSES/fonts` 폴더에서 확인하실 수 있습니다.

자세한 저작권 고지 및 배포 출처는 THIRD_PARTY_LICENSES/fonts 폴더의 각 문서를 참조하십시오.
